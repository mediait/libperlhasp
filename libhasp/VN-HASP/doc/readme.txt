До написания нормальной документации - вот такая тут цидулька

Для сборки дистрибутива нужна suse10.

Как установить всё:
1) Устанавливаем из redist RTE для хаспа:
rpm -e aksusbd-suse
rpm -iv aksusbd-1.15-1.i386-x86_64.rpm

2) Если необходимо, чтобы система работала только с хаспом с заданным
идентификатором, редактируем lib/VN/HASP/HASP4/VNHASP.h и 
lib/VN/HASP/HASPHL/VNHASP.h и ставим нужный идентификатор в константу HASPID.

3) Сборка модулей:
  make && make install

3.1) Для сборки модуля HASPHL на x86_64 необходимо установить libhasp_linux*.so
в каталог /usr/local/lib64. Makefile для модуля сам проверяет наличие 
установленных shared objects и в случае необходимости пытает их установить
(для этого используется 'sudo').

При этом модуле НЕ инсталлируются в /usr/lib/.../. Просиходит установка в 
локальный каталог ./lib.

4) Если надо установить модуль в другую папку lib:

  LIB=/path/to/lib_dir make dist

5) Если необходима утилита для записи чего-то в память хаспа, то можно 
использовать write_hasp_3 из каталога tools. Эта утилита использует модуль 
HASPTools, находящийся в том же каталоге (tools).
