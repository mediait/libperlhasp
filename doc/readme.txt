До написания нормальной документации - вот такая тут цидулька

Для сборки дистрибутива нужна suse10.

Как установить всё:
1) Устанавливаем из redist RTE для хаспа:
rpm -i aksusbd-suse-1.14-3.i386.rpm

2) Если необходимо, чтобы система работала только с хаспом с заданным
идентификатором, редактируем VN-HASP-HASP4/VNHASP.h и VN-HASp-HASPHL/VNHASP.h
и ставим нужный идентификатор в константу HASPID.

3) Собираем VN::HASP::HASP4 и устанавливаем:
cd VN-HASP-HASP4
make
make install

4) Собираем VN::HASP::HASPHL и устанавливаем:
cd VN-HASP-HASPHL
make
make install

5) Подкладываем в исходники вместо старого HASP.pm новый HASP.pm из каталога
VN.

6) Если необходима утилита для записи чего-то в память хаспа, то можно 
использовать write_hasp_3 из каталога tools. Эта утилита использует модуль
HASPTools, находящийся в файле HASPTools.pm из корневого каталога репозитария.
